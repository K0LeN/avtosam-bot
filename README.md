# ავტოსამრეცხაო Telegram Bot

## გაშვება Railway-ზე

### Environment Variables (გარემოს ცვლადები):
- `BOT_TOKEN` — Telegram bot token
- `GOOGLE_CREDENTIALS` — Google service account JSON (მთლიანი JSON ერთ სტრიქონად)
- `ALLOWED_USER_ID` — შენი Telegram User ID

### შენი Telegram ID-ის გასარკვევად:
გაუგზავნე @userinfobot-ს /start

## გამოყენება:
1. გაუგზავნე ბოტს ფოტო მანქანის
2. ბოტი დაგეკითხება ფასს
3. ჩაიწერება Google Sheets-ში ავტომატურად
